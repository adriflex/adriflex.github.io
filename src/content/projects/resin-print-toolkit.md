**Resin Print Toolkit** est un système de supports d'impression résine générés procéduralement dans Blender via Geometry Nodes.

Le problème classique de l'impression résine : générer des supports efficaces sans fragiliser le modèle ni gâcher la surface. L'approche procédurale permet d'ajuster finement densité, taille et emplacement des supports directement dans Blender — sans quitter son workflow habituel, sans logiciel de slicing intermédiaire pour cette étape.
